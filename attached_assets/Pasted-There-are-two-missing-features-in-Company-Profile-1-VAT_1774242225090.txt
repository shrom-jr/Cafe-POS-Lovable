1. VAT TOGGLE (GLOBAL)
========================

- In Settings (type + storage):
  Add:
    vatEnabled: boolean
    vatRate: number (default 13)

- Default values:
    vatEnabled = true
    vatRate = 13

- In Admin → Company Profile:
  Add a toggle switch:
    Label: "Enable VAT (13%)"

- Behavior:
  IF vatEnabled = true:
    - Show VAT row in receipt
    - VAT = (taxableAmount * vatRate / 100)

  IF vatEnabled = false:
    - DO NOT render VAT row at all
    - Total = taxableAmount (no VAT added)

- Apply this logic inside:
  ThermalReceiptLayout ONLY (single source)

- Ensure:
  - No undefined values
  - Safe fallback (vatEnabled ?? true)
